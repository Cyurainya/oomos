# osmos Atelier

Editorial React landing page for osmos, a small-edition knitwear atelier.

## Stack

- React
- Vite
- CSS animations and responsive layout
- Local image/video assets in `public/assets`

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Notes

Large campaign videos are committed under `public/assets`. If the repository grows much larger later, move video files to Git LFS or a CDN.
